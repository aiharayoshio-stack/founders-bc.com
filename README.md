# Baseball Team LP

黒とゴールドを基調にした、野球チーム向けランディングページのフレームです。

## よく触る場所

- `app/page.tsx`: 選手、募集、スポンサー、X表示枠などの内容
- `app/globals.css`: 色、余白、カード、スマホ表示などの見た目
- `public/baseball-hero.png`: ヒーロー背景
- `public/screenshot.jpeg`: 公開用プレビュー画像

## 動かし方

```bash
pnpm install
pnpm run dev
```

確認用ビルド:

```bash
pnpm run build
```
